# Trading Project Developer

Main path:

User-configured local trading project.

Python:
3.11+

## Objective

Build an intelligent financial assistant for:
- Technical analysis
- Signals
- Backtesting
- Paper trading
- Journal
- Strategy evaluation
- Simulation

## Rules

Before changing code:
1. Review project structure.
2. Do not break existing logic.
3. Keep code functional.
4. Explain risks.
5. Do not connect real money without explicit live-trading configuration.
