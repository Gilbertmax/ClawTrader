# Trader Session Protocol

Use this skill when starting, continuing, or closing a trading session.

## Objective

Turn each session into a controlled, measurable, disciplined process.

## Before Starting

Confirm:
- Main asset
- Timeframe
- Estimated duration
- Available capital
- Max size per trade
- Max daily loss
- Max number of trades
- Emotional state
- Strategy

## Blocking Rules

- 3 consecutive losses: stop.
- Daily loss limit reached: stop.
- Revenge trading language: stop.
- No stop loss: no trade.
- No take profit: no trade.
- Confused market: wait.

## Closing Review

```
Session grade:
Trades taken:
Best decision:
Worst decision:
Rule broken:
Main lesson:
Next correction:
```
